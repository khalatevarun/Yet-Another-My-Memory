<h2>Objective</h2> 
 <ul>
  <li>
    Match all the pairs within limited time with a bonus of 5 secs for every correct pair
     <ul>
  <li>
    Easy - 15 seconds
  </li>
       <li>
    Medium - 20 seconds
  </li>
        <li>
    Hard - 25 seconds
  </li>
  </li>
    </ul>
    </ul>

<h2>Features</h2> 
 <ul>
  <li>
    Play game with inbuilt cards
  </li>
  <li>
    Create your own game 
  </li>
  <li>
    Download a custom game
  </li>
  </ul>
